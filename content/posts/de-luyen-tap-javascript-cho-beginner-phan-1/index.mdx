## Tổng Quan

Phần đầu tiên trong bộ đề luyện tập JavaScript dành cho người mới bắt đầu học lập trình. Chúng ta sẽ tập trung vào các khái niệm nền tảng: biến, kiểu dữ liệu và toán tử. Thông qua các bài tập thực hành đơn giản, bạn sẽ làm quen và hiểu rõ các khái niệm cơ bản nhất trong JavaScript.

> **Luyện tập là cách tốt nhất để thành thạo JavaScript!**

## Phần 1: Biến, Kiểu dữ liệu và Toán tử

### Kiến thức cần nhớ

- **Biến (Variable)** là một tên gọi được gắn cho một vùng nhớ chứa dữ liệu. Dữ liệu này được gọi là giá trị. Có thể truy nhập, gán hoặc thay đổi giá trị của biến; khi gán giá trị mới, giá trị cũ sẽ bị ghi đè. Biến cần được khai báo trước khi sử dụng, sử dụng từ khóa `var`. Tên biến phải bắt đầu bằng một ký tự chữ cái (a-zA-z_), có thể theo sau bởi chữ, số.... Tên biến có phân biệt chữ hoa và chữ thường. Trong lập trình, chúng ta thường dùng các tên biến phổ biến như `name`, `age`, `x`, `y`, `count`, `total`, `price`, `result`.

- **Hằng (Constant)** là một tên gọi đại diện cho một giá trị cố định và giá trị của hằng không thể thay đổi. Giá trị của hằng cần được gán tại thời điểm khai báo, sử dụng từ khóa `const`.

- **Kiểu dữ liệu (Data Type)** là cách phân loại dữ liệu để trình biên dịch/thông dịch hiểu cách lập trình viên muốn sử dụng dữ liệu. Kiểu dữ liệu mô tả loại dữ liệu sẽ được lưu trong biến. Các kiểu dữ liệu thông dụng bao gồm: số (số nguyên, số thực, số dương, số âm), chuỗi, ký tự, và logic (true/false). Trong JavaScript, khi khai báo biến và gán giá trị, kiểu dữ liệu của biến đó được xác định.

- **Toán tử (Operator)** là các ký hiệu được sử dụng để thực hiện các thao tác trong biểu thức và sinh ra kết quả. Các loại toán tử thông dụng bao gồm: toán học, gán, cộng chuỗi, so sánh, logic, và typeof. Toán tử `typeof` được dùng để lấy về kiểu dữ liệu của một biến hoặc giá trị.

### Bài tập

#### 1. Khai báo biến và hằng

- Khai báo một biến `name` kiểu chuỗi và gán tên của bạn vào đó.
- Khai báo một biến `age` kiểu số nguyên và gán tuổi của bạn.
- Khai báo một hằng `PI` với giá trị 3.14.
- In giá trị của `name`, `age`, và `PI` ra console.

```javascript
// Khai báo biến và hằng
var name = "WanBi";
var age = 30;
const PI = 3.14;

// In ra console
console.log("Tên:", name);
console.log("Tuổi:", age);
console.log("Giá trị PI:", PI);
```

#### 2. Thay đổi giá trị và kiểm tra kiểu dữ liệu

- Gán lại giá trị cho biến `age` thành tuổi tiếp theo. In giá trị mới ra console.
- Sử dụng toán tử `typeof` để kiểm tra kiểu dữ liệu của `name` và `age`. In kết quả ra console.

```javascript
// Gán lại giá trị
age = age + 1;    // Cách đầy đủ
// hoặc 
age += 1;         // Cách viết gọn hơn
console.log("Tuổi sau sinh nhật:", age);

// Kiểm tra kiểu dữ liệu
console.log("Kiểu dữ liệu của name:", typeof name);  // Kết quả: string
console.log("Kiểu dữ liệu của age:", typeof age);    // Kết quả: number
```

#### 3. Sử dụng toán tử

- Khai báo hai biến số `x = 10` và `y = 3` (tên biến ngắn gọn, phổ biến cho các biến số).
- Thực hiện các phép toán cộng, trừ, nhân, chia, chia lấy phần dư (%) giữa `x` và `y`. In kết quả của mỗi phép toán ra console.
- Sử dụng toán tử tăng (`++`) cho `x` và toán tử giảm (`--`) cho `y`. In giá trị mới của `x` và `y` ra console.
- Sử dụng toán tử gán kết hợp (`+=`, `-=`, `*=`, `/=`, `%=`) cho `x` với giá trị 5. In kết quả sau mỗi phép gán ra console.
- Tạo hai biến boolean `is24HourTimeFormat = true` và `isDarkModeEnabled = false` (giống cài đặt trên iPhone). Sử dụng toán tử logic `&&` (AND), `||` (OR), `!` (NOT) với hai biến này. In kết quả ra console.

```javascript
// Khai báo biến số
var x = 10;
var y = 3;

// Thực hiện phép toán cơ bản
console.log("x + y =", x + y);    // Kết quả: 13
console.log("x - y =", x - y);    // Kết quả: 7
console.log("x * y =", x * y);    // Kết quả: 30
console.log("x / y =", x / y);    // Kết quả: 3.3333...
console.log("x % y =", x % y);    // Kết quả: 1 (phần dư của 10 ÷ 3)

// Sử dụng toán tử tăng/giảm
x++;  // tăng x lên 1
y--;  // giảm y đi 1
console.log("x sau khi tăng:", x);  // Kết quả: 11
console.log("y sau khi giảm:", y);  // Kết quả: 2

// Sử dụng toán tử gán kết hợp
x += 5;  // x = x + 5
console.log("x += 5:", x);  // Kết quả: 16

x -= 5;  // x = x - 5
console.log("x -= 5:", x);  // Kết quả: 11

x *= 2;  // x = x * 2
console.log("x *= 2:", x);  // Kết quả: 22

x /= 2;  // x = x / 2
console.log("x /= 2:", x);  // Kết quả: 11

x %= 5;  // x = x % 5
console.log("x %= 5:", x);  // Kết quả: 1
```

## Hướng dẫn thực hành

### Cách thực thi code JavaScript

1. **Mở trình duyệt web** (Google Chrome hoặc Firefox là lựa chọn phổ biến)
2. **Mở Developer Console** bằng cách:
   - Nhấn phím tắt `F12` hoặc `Ctrl+Shift+J` (Windows/Linux) hoặc `Cmd+Option+J` (Mac)
   - Hoặc chuột phải vào trang web trống và chọn "Inspect" > "Console"
3. **Copy từng đoạn code** và dán vào Console
4. **Nhấn Enter** để thực thi và xem kết quả ngay lập tức

### Phương pháp thực hành hiệu quả

1. **Thực hành từng bước:** Thực hiện lần lượt từng bài tập theo hướng dẫn
2. **Thử nghiệm:** Thay đổi giá trị của các biến và dự đoán kết quả trước khi chạy code
3. **Ghi chú:** Viết lại ý hiểu của bạn về mỗi khái niệm sau khi thực hành

### Một số ví dụ thực tế

- Tính điểm trung bình: `var score1 = 8.5; var score2 = 7; var average = (score1 + score2) / 2;`
- Kiểm tra điều kiện: `var pass = average >= 5; console.log("Đạt:", pass);`
- Tính chiết khấu giá sản phẩm: `var price = 100; var discount = 10; var finalPrice = price - (price * discount / 100);`

### Gợi ý mở rộng

- Thử sử dụng `let` thay vì `var` để khai báo biến và tìm hiểu sự khác biệt
- Sử dụng `typeof` để kiểm tra các giá trị khác như `[]` (mảng), `{}` (đối tượng), `null`, `undefined`
- Thử nghiệm với bài toán tính tiền tip: `var bill = 200; var tipPercent = 15; var tipAmount = bill * (tipPercent/100);`

> **Lưu ý:** Hãy lưu lại các đoạn code đã thực hành vào một file .js riêng để ôn tập sau này!
