## Giới thiệu

Trong lập trình hướng đối tượng, "lớp và đối tượng trong Java" là hai khái niệm nền tảng mà bất kỳ ai học Java đều phải nắm vững. Bài viết này sẽ giúp bạn hiểu rõ bản chất, cách sử dụng lớp và đối tượng, cũng như áp dụng vào các bài toán thực tế. Sau khi đọc xong, bạn sẽ tự tin hơn khi xây dựng các chương trình Java theo phong cách OOP.

## Nội dung chính

### Lớp trong Java là gì?

Lớp (class) là khuôn mẫu để tạo ra các đối tượng. Nó định nghĩa thuộc tính (biến) và hành vi (phương thức) của một nhóm đối tượng cùng loại.

```java
// Định nghĩa lớp Person
public class Person {
    String name;
    int age;

    void sayHello() {
        System.out.println("Xin chào, mình là " + name);
    }
}
```

### Đối tượng trong Java là gì?

Đối tượng (object) là thực thể được tạo ra từ lớp. Mỗi đối tượng có trạng thái riêng và có thể thực hiện các hành động do lớp định nghĩa.

```java
// Tạo đối tượng từ lớp Person
Person wanbi = new Person();
wanbi.name = "WanBi";
wanbi.age = 25;
wanbi.sayHello(); // Kết quả: Xin chào, mình là WanBi
```

### Tại sao lớp và đối tượng trong Java lại quan trọng?

- Giúp tổ chức mã nguồn rõ ràng, dễ bảo trì
- Tái sử dụng mã thông qua kế thừa và đa hình
- Mô hình hóa các thực thể ngoài đời thực vào chương trình

#### Ví dụ mở rộng: Quản lý danh sách học viên

```java
import java.util.ArrayList;

public class Student {
    String name;
    int id;

    Student(String name, int id) {
        this.name = name;
        this.id = id;
    }
}

public class Main {
    public static void main(String[] args) {
        ArrayList<Student> students = new ArrayList<>();
        students.add(new Student("Lan", 1));
        students.add(new Student("Minh", 2));
        for (Student s : students) {
            System.out.println(s.name + " - " + s.id);
        }
    }
}
```

### Một số tips khi làm việc với lớp và đối tượng trong Java

- Đặt tên lớp bằng danh từ, viết hoa chữ cái đầu
- Đối tượng nên được tạo bằng từ khóa `new`
- Sử dụng constructor để khởi tạo giá trị ban đầu cho đối tượng
