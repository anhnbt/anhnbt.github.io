## Giới thiệu

Trong lập trình Java, "mảng và phương thức" là hai khái niệm cực kỳ quan trọng giúp bạn tổ chức và xử lý dữ liệu hiệu quả. Bài viết này sẽ hướng dẫn các bạn cách khai báo, sử dụng mảng và xây dựng phương thức trong Java, kèm ví dụ thực tế để bạn dễ hình dung. Sau khi đọc xong, bạn sẽ hiểu rõ cách kết hợp mảng và phương thức để giải quyết các bài toán lập trình thường gặp.

## Nội dung chính

### Mảng trong Java là gì?

Mảng là một cấu trúc dữ liệu cho phép lưu trữ nhiều giá trị cùng kiểu dữ liệu trong một biến duy nhất. Ví dụ, bạn có thể lưu danh sách điểm số của học sinh trong một mảng.

```java
int[] diemSo = {8, 9, 7, 10, 6};
System.out.println("Điểm đầu tiên: " + diemSo[0]);
```

**Giải thích:**

- `int[] diemSo` khai báo một mảng số nguyên.
- Các phần tử được truy cập qua chỉ số, bắt đầu từ 0.

### Phương thức trong Java là gì?

Phương thức (method) là một khối mã thực hiện một chức năng cụ thể, giúp tái sử dụng và tổ chức chương trình tốt hơn.

```java
public static int tinhTong(int[] arr) {
    int tong = 0;
    for (int i = 0; i < arr.length; i++) {
        tong += arr[i];
    }
    return tong;
}
```

**Giải thích:**

- Phương thức `tinhTong` nhận một mảng số nguyên và trả về tổng các phần tử.
- Sử dụng vòng lặp để duyệt qua từng phần tử của mảng.

## Kết hợp mảng và phương thức trong Java

Việc kết hợp mảng và phương thức giúp chương trình của bạn linh hoạt và dễ bảo trì hơn. Ví dụ, bạn có thể viết phương thức để tìm giá trị lớn nhất trong mảng:

```java
public static int timMax(int[] arr) {
    int max = arr[0];
    for (int i = 1; i < arr.length; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    return max;
}
```

### Một số tips khi làm việc với mảng và phương thức

- Luôn kiểm tra kích thước mảng trước khi truy cập phần tử để tránh lỗi.
- Tách các chức năng thành phương thức nhỏ giúp mã dễ đọc và kiểm thử.
