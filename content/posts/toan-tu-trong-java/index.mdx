## Giới thiệu

Toán tử trong Java là một phần không thể thiếu giúp bạn thao tác với dữ liệu, xây dựng logic và phát triển ứng dụng hiệu quả. Trong bài viết này, mình sẽ hướng dẫn các bạn cách sử dụng các loại toán tử phổ biến trong Java, kèm ví dụ thực tế và giải thích dễ hiểu. Sau khi đọc xong, bạn sẽ nắm vững cách vận dụng toán tử trong Java để giải quyết các bài toán lập trình hàng ngày.

## Nội dung chính

### Các loại toán tử trong Java

Java cung cấp nhiều loại toán tử khác nhau, mỗi loại phục vụ một mục đích riêng. Dưới đây là các nhóm toán tử quan trọng mà bạn cần biết:

#### 1. Toán tử số học

Các toán tử số học giúp thực hiện các phép tính cơ bản như cộng, trừ, nhân, chia, chia lấy dư.

```java
int a = 10, b = 3;
System.out.println("a + b = " + (a + b)); // 13
System.out.println("a - b = " + (a - b)); // 7
System.out.println("a * b = " + (a * b)); // 30
System.out.println("a / b = " + (a / b)); // 3
System.out.println("a % b = " + (a % b)); // 1
```

#### 2. Toán tử so sánh

Dùng để so sánh giá trị giữa các biến, trả về kết quả true/false.

```java
int x = 5, y = 8;
System.out.println(x > y);  // false
System.out.println(x < y);  // true
System.out.println(x == y); // false
System.out.println(x != y); // true
```

#### 3. Toán tử logic

Kết hợp các điều kiện, thường dùng trong câu lệnh điều kiện và vòng lặp.

```java
boolean a = true, b = false;
System.out.println(a && b); // false
System.out.println(a || b); // true
System.out.println(!a);     // false
```

### Sử dụng toán tử trong Java để giải quyết bài toán thực tế

Giả sử bạn cần kiểm tra một số có phải là số chẵn hay không:

```java
int number = 12;
if (number % 2 == 0) {
  System.out.println(number + " là số chẵn");
} else {
  System.out.println(number + " là số lẻ");
}
```

#### Mẹo nhỏ khi dùng toán tử trong Java

- Luôn chú ý đến kiểu dữ liệu khi thực hiện phép chia để tránh lỗi chia cho 0 hoặc kết quả không mong muốn.
- Sử dụng dấu ngoặc để đảm bảo thứ tự thực hiện phép toán đúng như mong muốn.
